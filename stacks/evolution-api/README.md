# Evolution API

> WhatsApp API for managing messages and bots.

A powerful API wrapper for WhatsApp, allowing the creation of bots and automated messaging systems.

## Environment Variables

| Variable | Example | Description |
| :--- | :--- | :--- |
| `EVOLUTION_DOMAIN` | `evo.rafhael.com.br` | Domain where the API will be accessible. |
| `AUTHENTICATION_API_KEY`| `YourSecretApiKey` | Global Master Key to manage the API instances. |
| `DATABASE_CONNECTION_URI` | `postgresql://user:pass@host:5432/evo` | Full connection string to Postgres. |
| `HOST_DATA_PATH` | `/home/rafhael/evolution_data` | Host folder to save WhatsApp sessions/instances. |
