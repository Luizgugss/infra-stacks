# n8n

> Workflow automation tool.

n8n allows you to build complex automations between different apps and APIs.

## Environment Variables

| Variable | Example | Description |
| :--- | :--- | :--- |
| `N8N_DOMAIN` | `n8n.rafhael.com.br` | Domain to access the n8n Editor. |
| `GENERIC_TIMEZONE` | `America/Sao_Paulo` | Timezone for workflows and schedules. |
| `HOST_DATA_PATH` | `/home/rafhael/n8n_data` | Host folder to persist workflows and credentials. |
